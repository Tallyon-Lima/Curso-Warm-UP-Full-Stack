Mapa Mental

Aula 1 - NoSQL

Por que utilizar o NoSQL para aplicações escaláveis e de alta performance?

Maior Armazenamento

Em casos de diversos usuários, o NoSQL permite armazenar grandes volumes de dados de forma mais eficiente. Como o exemplo de empresa de streaming de vídeos, no qual os usuários acabam gerando muitos dados simultaneamente.

Maior Desempenho

Com o NoSQL, é possível obter um desempenho superior em operações de leitura e escrita em comparação com os bancos de dados relacionais tradicionais.
